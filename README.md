# AyraTech Supermarket Manager

Sistema de gestão de supermercados, promotores e rotas.

## Módulos

- Backend (NestJS)
- Web Admin (React + Vite)
- Promoter PWA (React + Vite)

## Features

- Gestão de Clientes e Marcas
- Gestão de PDVs e Grupos
- Gestão de Promotores e Supervisores
- Roteirização e Checklist
- App do Promotor (PWA)
- Relatórios e Dashboard
- Controle de Validade e Rupturas
- Controle de Qualidade de Imagem (Anti-Fraude)
